# City Photogrammetry Showcase

This showcase demonstrates how to load and render a 3D city photogrammetry model using mapv-three and Three.js within a React component.

## Features

- Loading and rendering of 3D photogrammetry data
- Camera controls for exploring the 3D city model
- Optimized performance with screen space error management
- Compass navigation control

## Files

- `index.jsx`: Main entry point for the showcase
- `CityPhotogrammetry.jsx`: The main component implementing the city photogrammetry visualization
- `CityPhotogrammetry.less`: Styles for the showcase
